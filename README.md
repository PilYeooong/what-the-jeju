
# **What the Jeju**

**What the Jeju**는 제주 소개 커뮤니티 사이트입니다. 기존 Hot Jeju 프로젝트의 기능 확장판 입니다.

> **Hot Jeju**를 통해 구현해보았던 기능의 대한 구현은 보류한 상태로, 추가적으로 구현해보고자 하는 기능들을 우선적으로
> 구현하고 있습니다.
> 
>  - 외부 지도 API를 활용한 여행 동선 추천
>  - 개선된 UI / UX
>  - 모바일 반응형

## **Stack**
- Node.js
- Express
- MySQL
- React.js
- Redux, Redux-Saga
- Docker
